🐷 この対戦のルール

基本

  • 4人専用。通常カード52枚＋ババブタ1枚を使い、通常カード1枚を抜いて各自13枚。
  • 強さ：13 ＞ 12 ＞ … ＞ 1。1が最弱、13が最強。
  • リードスートがあれば必ず出す。ババブタは場に出せない。
  • 勝者はリードスートで一番強いカード。
  • 最弱は非リードスートがあればその中の最小。全員フォローなら場の最小。同数は後出し。

ピックとペア

  • 勝者は場の4枚をごちそう山へ。
  • 勝者は最弱の手札から1枚ピック。
  • 標準では、最弱側がピック候補を2枚に絞る。絞らない/1〜13枚にも変更可。
  • 最弱側の候補選択画面はスート順→数字順。勝者側のピック画面はランダム順。
  • 同じ数字ができたら、ペアで捨てるかスキップを選ぶ。ババブタはペア不可。

ラウンド終了

  • 手札0枚になったら終了条件。トリック中なら、そのトリック終了後にラウンド終了。
  • ババブタ1枚だけの場合は、その人の手番開始時にラウンド終了。
  • カードを出してババブタ1枚だけになっても即終了せず、必要なピック処理までは進みます。

得点・危険カード

  • スート：🍎リンゴ（赤）/ 🌽トウモロコシ（黄）/ 🥬キャベツ（緑）/ 💧ぬかるみ（灰）。
  • ごちそう山：+1点/枚。
  • 残り通常手札：標準はリンゴ・トウモロコシ・キャベツが1枚-3点、通常の💧が1枚-6点。全スート-3、数字分、または他-1・💧-3にも変更可。
  • マッド・ピッグ：💧11。標準ON。数字分以外では手札・山とも固有-13点で、通常カード失点と重複しない。数字分失点では手札なら-40点、山なら+1点後に-40点。
  • ババブタ：標準-20点。失点タイミングはラウンド毎/ゲーム最後から選択。
  • シュート・ザ・ピッグON時：ババブタ＋マッド・ピッグの両方が手札にある状態でラウンド終了すると発動。本人の対象失点0、他全員-10点。各プレイヤー1ゲーム1回までで、山のマッドは条件に含めない。

開始時オプション

  • 3枚パスON：次の手番の人へ3枚渡す。ババブタ不可。
  • 開始時ペア捨てON：同じ数字ペアを任意で捨てられる。

標準設定

  • 3ラウンド / ピック対象2枚 / リンゴ・トウモロコシ・キャベツは-3点・通常の💧は-6点。
  • マッド・ピッグあり / ババブタ-20点・ラウンド毎。
  • 3枚パスなし / 開始時ペア捨てなし / シュートON（両方とも手札・各自1回）。
