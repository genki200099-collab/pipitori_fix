# ピピとりオンライン（CPUコメント＆演出版）

## 現行の標準ルール（2026-07-20更新）

この節が現在の実装上の標準設定です。以下に残る「今回の修正」各節は変更履歴であり、標準ルールについて食い違う記述がある場合はこの節を優先してください。

- 3ラウンド
- ラウンド終了後は、全カードを回収してシャッフルし、全員へ新しい13枚を配る（標準）
  - 部屋作成時に「残りカードを持ち越して補充」へ変更可能
  - 全シャッフルでも各ラウンドのカード得点は累計する
- スートは🍎リンゴ（赤）／🌽トウモロコシ（黄）／🥬キャベツ（緑）／💧（灰）
- 残り手札はリンゴ・トウモロコシ・キャベツが1枚につき-3点、通常の💧が1枚につき-6点
- マッド・ピッグ（💧11）は手札・ごちそう山とも固有-13点。通常の💧失点とは重複しない
- ババブタは-20点・ラウンド毎
- シュート・ザ・ピッグは標準ON
  - ババブタとマッド・ピッグの両方が手札にある状態でラウンド終了すると発動
  - ごちそう山のマッド・ピッグは発動条件に含めない
  - 発動者の対象となるババブタ／マッド失点は0、他3人は各-10点
  - 各プレイヤーが発動できるのは1ゲームに1回まで
- ピック候補は2枚
- 最弱者への得点ボーナスはなし
- 開始時3枚パスなし、開始時ペア捨てなし

`npm test` で、標準値、各失点方式、マッド失点の非重複、シュートの手札条件・1回制限・複数ラウンド時の得点を回帰確認できます。
