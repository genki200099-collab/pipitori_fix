# 端末別レスポンシブ総合ブラッシュアップ

作成日時：2026-06-22 08:25 JST

## 方針
ゲームを盛り上げる演出は維持しつつ、iPhone・Android・PCでUIが破綻しないように整理。

## 修正内容
- iPhone/Android/タブレット/PC向けに `ui-phone` `ui-tablet` `ui-desktop` などの端末別クラスを自動付与。
- スマホ縦画面では場のカードを2x2に固定し、横はみ出しを防止。
- スマホ横画面では横長卓面に切り替え。
- PCでは情報量と演出を維持し、広い卓面で表示。
- タブレットではPCとスマホの中間レイアウトに調整。
- スマホではプレイヤー欄を2列にし、縦長化を抑制。
- 「次」矢印バッジ、PICK対象、WIN表示がカードに重なりすぎないようスマホ用に内部配置。
- safe-area と Safari/Androidブラウザ下部バーを考慮して下余白を追加。
- タッチ端末では大きな移動アニメーションだけ抑制し、演出は色・光・トースト中心に維持。
