# ロビーUI改善・全体点検

作成日時：2026-07-04 11:18 JST

## 改善内容

### 1. 「部屋を作る」ボタンの押下状態を分かりやすく
- 押した直後に「部屋を作成中…」へ変更
- 色をオレンジ系へ変更
- 接続中のアニメーションを追加
- 作成完了後に「部屋作成済み」へ変更
- 作成失敗/切断時は「もう一度作る」へ変更

### 2. 部屋作成後の導線を明確化
- 「部屋を作成しました」バナーを追加
- 部屋コードの上に説明を追加
- 現在の操作段階が分かる3ステップ表示を追加

### 3. 「4人で遊ぶ」ボタンを目立たせる
- ボタンを横長・中央配置へ変更
- 4人そろったら大きく目立つ色へ変更
- 人数不足時は「あとN人で開始」と表示
- 補助説明文を追加

### 4. UI全体の分かりやすさ
- CPU追加/削除とゲーム開始ボタンをひとまとまりの開始パネルとして整理
- スマホでも縦に自然に並ぶよう調整
