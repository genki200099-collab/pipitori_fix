# PC卓面UI整理・冗長説明削除

作成日時：2026-07-04 11:27 JST

## 原因
PCブラウザでは卓面が横長に広がるため、座席内に残っていた小さな順番矢印チップが座席の境界付近に出てしまい、矢印UIが崩れて見えていた。

スマホでは表示幅が狭いため目立ちにくかったが、PCでは座席枠・中央ホイール・順番バーと情報が重複していた。

## 修正内容
- 卓面内の座席ごとの小矢印チップを削除
- 順番表示は上部のプレイトレイルに統一
- PC横幅で座席枠と中央ホイールが崩れにくいよう、卓面グリッド幅を調整
- スマホ/極小幅の卓面レイアウトも維持
- 場のカード枠の冗長な説明文「出せるカードだけ光ります。迷ったら中央メッセージを見てください。」を削除
- 手札側の「光っているカードを選んでください。」は維持
