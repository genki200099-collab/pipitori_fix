# モダンゲームUIブラッシュアップ

作成日時：2026-07-04 00:17 JST

## 目的
最新のブラウザゲーム/ボードゲームアプリ風に、全体UIの見た目・情報階層・操作感を整理。

## 主な改良
- 背景をリッチなグラデーション＋薄いグリッドに変更。
- カードボックスをガラス調の面に変更し、画面階層を整理。
- ロビー、設定、ステータス、プレイヤー、卓面、手札、ルール早見表を統一感ある角丸カードUIに調整。
- ボタン、入力欄、部屋コード、設定チップの質感とタッチしやすさを改善。
- 卓面と手札カードの立体感を強化。
- スマホ/タブレット/PCで余白とカラム数を調整。
- シュート・ザ・ピッグ演出や既存のちらつき抑制は維持。

## 触っていないもの
- ゲーム進行ロジック
- 得点計算
- ピック処理
- シュート・ザ・ピッグ処理
- ババブタ失点タイミング
- Render Build設定

## 検証
- server.js 構文チェック
- クライアントJS構文チェック
- 既存重要仕様の文字列チェック
