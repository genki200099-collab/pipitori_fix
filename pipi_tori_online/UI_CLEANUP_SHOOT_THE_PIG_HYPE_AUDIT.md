# UI整理・シュート・ザ・ピッグ演出強化

作成日時：2026-07-03 23:52 JST

## UI整理
- 画面上部の設定チップを必要最小限に整理。
- OFF系の情報は原則表示せず、ON/変更された設定だけを優先表示。
- 「マッドあり」「シュートOFF」など、常時見なくてもよい情報を削減。
- 部屋作成時、マッド・ピッグOFFならシュート・ザ・ピッグを送信前にも必ずOFF同期。

## シュート・ザ・ピッグ演出
- シュート・ザ・ピッグ発動時に専用オーバーレイ演出を追加。
- 発動者名、効果「ババブタ失点0 / マッド失点0 / 他全員-10」を大きく表示。
- 放射背景、ポップ、発光、紙吹雪風の演出を追加。
- ラウンド結果画面の発動表示も専用バナー化。

## ちらつき配慮
- 既存のラウンド結果/最終結果キャッシュを維持。
- シュート演出はイベントID単位で一度だけ表示。
- overlayは一定時間後に自動でフェードアウト。
- 既存のピック・手札・卓面の再描画抑制は維持。

## 維持した仕様
- ピック対象数デフォルト2枚。
- 最弱側の候補選択はスート順→数字順。
- 勝者側のピック表示はランダム順。
- ババブタ失点タイミング設定。
- シュート・ザ・ピッグの得点処理。
- Render Build対策。
