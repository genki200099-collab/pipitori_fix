# ピック対象数デフォルト2枚化

作成日時：2026-07-03 23:34 JST

## 修正内容
部屋作成時の「ピック対象数」のデフォルトを「絞らない」から「2枚」に変更。

## 変更箇所
- 部屋作成画面の初期選択を「2枚」に変更
- クライアント送信時のフォールバック値を2に変更
- サーバー側 createRoom() のデフォルト値を2に変更
- 表示文内のデフォルト表記を可能な範囲で更新

## 維持した仕様
- 「絞らない」選択肢は残す
- 1〜13枚の選択肢は維持
- 最弱側の候補選択画面はスート順→数字順
- 勝者側のピック画面はランダム順
- シュート・ザ・ピッグ、ババブタ失点タイミング、マッド・ピッグ設定は維持
