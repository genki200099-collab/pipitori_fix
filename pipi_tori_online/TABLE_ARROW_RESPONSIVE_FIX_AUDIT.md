# 卓面矢印UIのPC/スマホ最適化

作成日時：2026-07-04 01:05 JST

## 原因
旧UIでは、各座席の矢印 `.seat-arrow` を `position:absolute` で座席の外側へ出していました。

例：
- 1番席：`right:-24px`
- 2番席：`bottom:-22px`
- 3番席：`left:-24px`
- 4番席：`top:-22px`

そのため、PCブラウザの横長レイアウトでは、矢印が隣の座席や中央の時計回り表示へはみ出し、添付画像のように崩れていました。

## 修正内容
- 座席外へ出す `.seat-arrow` を廃止。
- 座席内に収まる `.seat-flow-chip` に変更。
- PC、タブレット、スマホでそれぞれサイズと配置を調整。
- PCでは広い3カラム卓面を維持しつつ、矢印を座席内右下に固定。
- スマホでは矢印を丸アイコン中心にして、カードや名前を邪魔しないよう圧縮。
- 極小幅では中央ホイールを上段に逃がし、卓面を2列表示に変更。

## 維持した仕様
- 時計回りの順番表示
- 上部のプレイトレイル表示
- トリック結果確認中のカウントダウン
- ピック画面のちらつき抑制
- ゲームロジック、得点計算、CPU戦略
