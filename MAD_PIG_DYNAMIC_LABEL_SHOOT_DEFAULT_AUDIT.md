# マッド・ピッグ可変表示・シュート標準ON点検

作成日時：2026-07-18 JST

## 変更内容

- マッド・ピッグのカード上の失点チップを得点方式に連動。
  - `faceValue`：`失点 -40`
  - `flat3` / `spadeSuit`：`失点 -13`
- シュート・ザ・ピッグの標準設定をONへ変更。
- ピック候補確定のWebSocket受信分岐が1回だけ実行されることを確認。

## 整合確認

- 部屋作成画面の初期選択：ON
- 標準設定の説明：ON
- ルール早見表：ON
- サーバーの引数省略時既定値：ON
- マッド・ピッグOFF時：従来どおりシュートを自動OFF
- 得点計算とCPU判断：既存のシュート処理を維持
- マッド・ピッグのカード表示：選択された失点方式に追従
- 引き継ぎ仕様書：デフォルトONへ更新
- ピック候補確定要求：受信分岐は1つ

## 維持した仕様

- 数字分失点モードのマッド・ピッグは手札なら-40点、ごちそう山なら+1点後に-40点。
- 1枚-3点モードと♠-3/他-1モードではマッド・ピッグ失点-13点。
- ♠-3/他-1モードでは通常の♠失点-3とマッド失点を二重加算しない。
- シュート・ザ・ピッグはマッド・ピッグON時のみ有効。
